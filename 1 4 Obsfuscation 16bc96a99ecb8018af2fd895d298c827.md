- **Steganography**
    - make message invisible in a container
        - in image
        - in tcp packet
        - invisible watermark
        - in audio / video
- **Tokenization**
    - replace data w/ single use token (ex temporary “one time use” credit card )
    - replace token w/ data at the end of the process
